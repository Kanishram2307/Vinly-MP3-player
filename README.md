# Vinly-MP3-player
A MP3 player created using Python to fulfill my Hobby as an Audiophile 
# Vinyl

A minimal black-themed Android music player built using KivyMD.

## Features
- Play/Pause
- Previous/Next
- Shuffle
- Loop
- File Picker
- Folder Picker
- Monthly Wrap (Spotify-style statistics)
- Audio Output Detection

## Technologies
- Python
- Kivy
- KivyMD
- SQLite

## Installation

pip install -r requirements.txt
python main.py
