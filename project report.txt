Abstract:
VINYL is a minimal black-themed Android music player developed using Python, Kivy, and KivyMD. The application enables users to play audio files stored on their devices while providing a clean and modern user interface. The application supports file and folder selection, playback controls, shuffle mode, loop mode, monthly listening statistics, and audio output device detection.
The main objective of this project is to create a lightweight and user-friendly music player that offers essential playback features along with personalized listening insights similar to modern streaming applications.

Chapter 1: Introduction
Music players are among the most frequently used mobile applications. Many existing music players include unnecessary features that consume storage and system resources. VINYL was developed to provide a lightweight and efficient alternative with a clean user experience.
The application allows users to access locally stored music files and manage playback with simple controls. It also records listening history and generates monthly summaries of frequently played tracks.

Chapter 2: Objectives
The objectives of the project are:

Chapter 4: System Architecture
The system consists of the following modules:
User Interface Module
Provides buttons and controls for music playback.
File Manager Module
Allows users to select music files or folders from device storage.
Audio Playback Module
Loads and plays audio files using Kivy's SoundLoader.
Statistics Module
Stores playback history in SQLite and generates monthly listening reports.
Audio Detection Module
Identifies connected output devices such as headphones and Bluetooth speakers.

Chapter 5: Methodology
User selects a file or folder.
Application scans supported audio formats.
Playlist is generated.
Selected song is loaded.
User controls playback through UI buttons.
Playback history is stored in SQLite database.
Monthly statistics are generated from stored data.
